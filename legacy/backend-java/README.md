# Java 原型归档

当前目录保存的是第一轮 `Java + Spring Boot` 规则验证原型。

从现在开始：

- 这个目录不再作为正式主线继续扩展
- 只用于保留已经验证过的业务规则、测试样例和数据库结构草稿
- 新的正式执行基线已经切换到：
  - `apps/api`
  - `apps/ai-runtime`
  - `apps/mcp-gateway`
  - `apps/worker`

如需查看新的执行基线，请先看：

- [后端技术栈重决策](../../docs/architecture/backend-tech-stack-redecision.md)
- [后端重构方案](../../docs/architecture/backend-architecture-redesign.md)
